# Proyecto Final Antonio Gutíerrez

_API RESTfull para la gestion de clientes, paises, productos y pedidos_

## Comenzando 🚀

_Estas instrucciones te permitirán obtener una copia del proyecto en funcionamiento en tu máquina local para propósitos de desarrollo y pruebas._

Mira **Deployment** para conocer como desplegar el proyecto.


### Pre-requisitos 📋

_Necestias tener instaldo Java 17_

```
www.java.com/es/download

```

### Instalación 🔧

_Descargate el repositorio directamente o copialo mediante el siguiente comando_

_Dí cómo será ese paso_

```
git clone https://github.com/memecito/antonio-gutierrez-final-project.git
```

_abre una terminal y en la raiz del repositorio ejecuta_

```
./mvnw clean package
```
